# User-Tech

Personal productivity tools for local workflows.

## Setup
`pip install -r user-tech/requirements.txt`

## Tools
- **url-crawl-to-docx/**: crawl same-domain pages and export text to DOCX (guardrails included)
- **pst-to-eml/**: convert PST archives to EML using readpst (external dependency)

These tools can generate large outputs. Test with small inputs first.

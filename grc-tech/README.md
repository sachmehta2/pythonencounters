# GRC-Tech

Security/GRC inspection and evidence-style helpers.

## Setup
`pip install -r grc-tech/requirements.txt`

## Tools
- **email-header-analyzer/**: email header triage (SPF/DKIM/DMARC hints)
- **ssl-cert-check/**: certificate metadata inspection (inspection-only)
- **net-monitor/**: local network connection logging (authorized systems only)
- **file-hash-md5/**: MD5 integrity receipt for a file

Run only on systems you own/are authorized to inspect.
